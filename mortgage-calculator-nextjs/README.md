# Ипотечный калькулятор (Next.js)

Функции:
- Аннуитетный платёж
- Дифференцированный платёж
- Таблица амортизации + экспорт CSV
- Подбор условий (аннуитет): макс. сумма под платёж, нужная ставка, нужный срок (с лимитом)

## Запуск

1) Поставь Node.js 18+ (лучше 20 LTS)
2) В папке проекта:

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Публикация

Самый простой вариант: GitHub → Vercel → Deploy.
